Upload a JSON file per language and it will be converted directly to chunks in the knowledge base. Accepts arrays of objects, plain text arrays, or smart-scraper output format.

Drop JSON file here or browse
One JSON file per language recommended
Accepted formats:
[{"text":"product info...","product_name":"..."}, ...] ← object array
["chunk text 1", "chunk text 2"] ← string array
{"products":[...]} ← smart scraper output
Scrape & Process

Scrapes all configured URLs, saves raw text files, then processes them into chunks ready for translation and upload.

Smart Scrape — uses Groq LLaMA to extract clean structured product data automatically. No manual editing needed.
Raw Scrape — saves raw page text as .txt; use Process → Chunks afterwards.
